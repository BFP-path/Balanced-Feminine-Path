# Balanced Feminine Path 🌿

A sanctuary for women seeking to align with their true selves through holistic wellness, mindful practices, and intentional living.

## About

Balanced Feminine Path is a modern website built with [Astro](https://astro.build) — designed to be fast, beautiful, and SEO-friendly.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📁 Project Structure

```
├── src/
│   ├── layouts/          # Page layouts
│   └── pages/           # Pages (Home, Blog, About, Resources, Contact)
├── public/              # Static assets
├── astro.config.mjs     # Astro configuration
└── package.json
```

## ☁️ Deploy to Vercel

1. Push to GitHub
2. Connect to [Vercel](https://vercel.com)
3. Add your domain: `balancedfemininepath.com`

## ✨ Features

- ⚡ Fast, static site generation
- 📱 Fully responsive design
